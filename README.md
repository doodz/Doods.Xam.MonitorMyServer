# Doods.Xam.MonitorMyServer

Monitor my server is a project written in C# (Xamarin.Forms) for android and windows UWP.
First step I want manage Raspberry pi and openmediavault in ssh
For :
  * Check disk space.
  * System informations.
  * Updates.
  * View running processes
  * Run commands.
